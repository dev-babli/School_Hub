Smart Attendance – Setup Guide

Follow these steps to run the Smart School Attendance system (face recognition + WhatsApp alerts).

1. Open the project folder

cd School_Hub

2. Start the frontend (Terminal 1)

cd School_Hub
npm install
npm run dev

Dashboard: http://localhost:3000
Go to http://localhost:3000/admin/dashboard

3. Install Python dependencies (one-time)

cd School_Hub\face-recognition-poc
pip install opencv-python opencv-contrib-python requests

4. Enroll faces (Terminal 2)

cd School_Hub\face-recognition-poc
python enroll_face.py
  • Press Enter for Hikvision camera (or type d for DroidCam)
  • Enter name (e.g. Asif)
  • Enter parent WhatsApp number (e.g. 916371070959)
  • Follow on-screen instructions: look at camera, turn left, turn right
  • The script will remind you: tell the parent to send join worth-on to +1 415 523 8886

5. Parents join WhatsApp sandbox (to receive messages)

Parents must do this once before they can receive attendance messages:

  • Open the dashboard: http://localhost:3000/admin/dashboard
  • In the Demo Setup card, find the QR code and scan it with WhatsApp, OR
  • From their WhatsApp, send: join worth-on to +1 415 523 8886

6. Start attendance (Terminal 2)

cd School_Hub\face-recognition-poc
python attendance_poc.py

The camera will start. When a recognized face appears, attendance is logged and WhatsApp is sent.
